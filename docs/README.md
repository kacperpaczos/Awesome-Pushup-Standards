# Documentation site

The project wiki is built with [Astro Starlight](https://starlight.astro.build/) in [`apps/docs/`](../apps/docs/).

```bash
npm run docs:dev     # sync + local preview → http://localhost:4321
npm run docs:build   # sync + production build → apps/docs/dist/
npm run docs:sync    # refresh plugin/preset pages from packages/*/README.md
```

- **Guides** — hand-written in `apps/docs/src/content/docs/guides/`
- **Plugins / Presets** — generated by `scripts/sync-docs-to-starlight.mjs`
- **Registry** — [documentation-registry](../apps/docs/src/content/docs/reference/documentation-registry.md) (generated; run `npm run docs:sync`)

CI uploads `apps/docs/dist/` as artifact `docs-dist`.
