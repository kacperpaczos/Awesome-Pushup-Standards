# Documentation

Site documentation lives in **[Starlight](https://starlight.astro.build/)** at [`apps/docs/`](../apps/docs/).

## Local preview

```bash
npm run docs:dev
```

Open [http://localhost:4321](http://localhost:4321).

## Build

```bash
npm run docs:build
```

Static output: `apps/docs/dist/`.

## Source files

Markdown content: [`apps/docs/src/content/docs/`](../apps/docs/src/content/docs/).
